This project analyzes air pollution levels across major cities and states in India. It identifies the most and least polluted regions and examines how individual pollutants contribute to overall air quality. The goal is to provide clear, data‑driven insights into pollution patterns and pollutant impact.

📌 Objectives
Rank cities and states by pollution levels

Identify the cleanest and most polluted regions

Analyze pollutant contributions (PM2.5, PM10, NO₂, SO₂, CO, O₃)

Visualize pollution distribution and trends

📂 Project Structure
Code
data/          → Raw and cleaned datasets
notebooks/     → Jupyter notebook for analysis
scripts/       → Python scripts for cleaning & visualization
outputs/       → Generated charts and results
README.md
📊 Key Insights
PM2.5 and PM10 are the major contributors in most cities

Several states show consistently high AQI levels

Least polluted cities exhibit lower particulate matter and stable gas pollutant levels

🛠️ Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook
